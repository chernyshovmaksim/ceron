Режимы создания

Обычный

Создание обычного DIV элемента.

Секция

SECTION элемент с контейнером и заголовком внутри.

Элемент

Создает DIV элемент с эффектами.

Текста

Работа с текстами, создает и парсит текст.

Список

UL список с LI элементами.

Меню

Вертикальное или горизонтальное меню.

Колонки

Bootstrap колонки.

Кнопка

Создает кнопку с текстом и эффектами.

Input

Создает поле ввода с эффектами.

Textarea

Создает текстовое поле с эффектами.

Шаблон

Создает из выделенных элементов шаблон.

В Шаблон

Переводит выделенные элементы в шаблон.

Форма

Создает поля, кнопки и текста.

Иконка

Создает DIV с фоновой картинкой.

IconBox Horisontal

FLEX элемент с иконкой и текстом внутри.

IconBox Vertical

FLEX элемент с иконкой и текстом внутри.
